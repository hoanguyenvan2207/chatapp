# chatapp

Ứng dụng chat realtime đơn giản dùng **Express + Socket.IO**.

## Cài đặt

```bash
npm install
```

## Chạy app

```bash
npm start
```

Mở trình duyệt tại `http://localhost:3000`.

## Tính năng

- Chat realtime nhiều người dùng.
- Hiển thị thông báo vào/ra phòng.
- Tự nhập tên người dùng khi vào.
- API kiểm tra trạng thái: `GET /health`.
